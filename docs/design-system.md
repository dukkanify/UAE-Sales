# Sooqna Premium Design System

## Brand Direction

Sooqna should feel like a premium UAE marketplace: trustworthy, polished, investor-ready, and built for high-value transactions. The visual language should be closer to modern real-estate, fintech, and hospitality products than a generic classifieds template.

## Color Palette

| Role | Token | Hex | Usage |
| --- | --- | --- | --- |
| Primary | Charcoal | `#111827` | Navigation, primary text, primary buttons, premium dark surfaces |
| Secondary | Gold | `#C8A45D` | Premium highlights, badges, key CTAs, borders, trust marks |
| Accent | UAE Red | `#B91C1C` | UAE identity accent, alerts, small brand strips, destructive actions |
| Background | Warm Light | `#F8F7F4` | Page background and large sections |
| Surface | Ivory White | `#FFFFFF` | Cards, forms, panels |
| Muted Surface | Warm Sand | `#EFEAE1` | Secondary panels, input backgrounds |
| Text Muted | Warm Grey | `#6B665C` | Descriptions, helper text |
| Border | Soft Stone | `#E5E7EB` | Card borders, input borders, dividers |

Green is intentionally removed from the UI identity. UAE influence is expressed through premium charcoal, gold, and a restrained red accent.

## Typography

- Font stack: `Tajawal`, `IBM Plex Sans Arabic`, `Tahoma`, `Arial`, sans-serif.
- Hero titles: very bold, tight tracking, generous line height for Arabic readability.
- Section titles: bold and editorial, 32-44px depending on viewport.
- Body copy: 16-18px with relaxed line height.
- UI labels/buttons: 13-15px, black weight for confidence and clarity.

## Layout & Spacing

- Container width: 1240px.
- Page background: warm light with subtle radial highlights.
- Sections: 64-96px vertical spacing on desktop, 40-56px on mobile.
- Cards: large radius, warm borders, soft shadows, clear hierarchy.
- Mobile: single-column first, no horizontal overflow, large touch targets.

## Components

### Buttons

- Primary: charcoal background, gold hover/accent edge, strong contrast.
- Secondary: white/warm surface, charcoal text, gold border on hover.
- Ghost: transparent, muted text, warm hover.

### Cards

- Warm white surfaces, stone borders, soft shadows.
- Listing cards should look like premium inventory cards, not generic tiles.
- Category cards use gold icon containers and restrained red strips.

### Badges

- Gold-tinted badges for premium/trust.
- Red only for UAE accent or warnings.
- Avoid bright saturated colors except small accent details.

### Forms

- Warm white inputs, generous height, clear Arabic labels.
- Validation errors in UAE red.
- Auth forms include OTP readiness and trust messaging.

## Homepage Structure

1. Premium hero with UAE marketplace positioning.
2. Prominent search panel.
3. Category grid with premium cards.
4. Featured listings with upgraded listing cards.
5. Escrow protection section.
6. Trust and safety section.
7. Footer with organized links.

## Interaction & UX Principles

- No broken links.
- Clear empty states.
- Visible trust cues around payment, seller identity, and escrow.
- Main flow must work with mock/local data: register, login, add listing, view listing, search listing, dashboard.

## Visual Rules

- Do not use green as the brand color.
- Keep red restrained; it should not dominate.
- Use gold as the premium signal.
- Avoid template-like icon overload or random gradients.
- Maintain Arabic RTL correctness across all layouts.
