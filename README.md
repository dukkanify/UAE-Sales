# UAE Sales Web

واجهة ويب عربية RTL لسوق إعلانات مبوبة في الإمارات، مبنية باستخدام Next.js وTypeScript وTailwind CSS.

## التشغيل

```bash
npm install
npm run dev
```

افتح `http://localhost:3000`.

## البنية

- `app` صفحات وتخطيط Next.js.
- `components` مكونات الواجهة القابلة لإعادة الاستخدام.
- `layouts` عناصر التخطيط العامة مثل Header وFooter.
- `services` طبقة البيانات والـ API.
- `hooks` Hooks جاهزة لتفاعلات الواجهة.
- `types` أنواع TypeScript المشتركة.
- `constants` ثوابت التنقل والمواقع.
- `assets` أصول المشروع.
- `styles` متغيرات التصميم والملفات العامة.
